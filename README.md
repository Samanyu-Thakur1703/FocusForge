# FocusForge

FocusForge is a student productivity MVP built with Next.js App Router, Supabase, Tailwind CSS, shadcn/ui, Zustand, and Zod.

This repository is currently scaffolded through the foundation phase only:

- Next.js project tooling
- Tailwind CSS and shadcn/ui configuration
- TypeScript path aliases
- Environment variable handling
- Supabase client, server, and middleware foundation
- Route, feature, component, store, validation, and test placeholders

Business logic, database schema, protocol generation, and feature implementation come in later phases.
